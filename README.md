# GITHUB

- Para agregar archivos 
```
git add nombre_Archivo
```

- Para agregar todos los archivos de la carpeta actual
```
git add .
```

- Para hacer commit de los archivos
```
git commit -m "first commit"  
```

- Para subir archivos
```
git push -u origin main  
```
